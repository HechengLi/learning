webdav is a set of extensions to http protocol
it allows user agents to collaboratively author contents directly in an http web server
actions like create/copy/paste/move, encrypt/decrypt files
webdav methods
PROPFIND
PROPPATCH
MKCOL
COPY
MOVE
LOCK
UNLOCK
